# ROSTER ALGORITHM
Simple Algorithm to Generate Roster for your team

## DEMO
https://plnkr.co/edit/kk7tcf?p=preview


## DATA

Create a object with members name in below structure

```JSON
{

  "members": [{
      "id": 1,
      "name": "John",
      "team": 1
    },
    {
      "id": 2,
      "name": "Paul",
      "team": 2
    },
    {
      "id": 3,
      "name": "Simon",
      "team": 1
    }]
}

```
